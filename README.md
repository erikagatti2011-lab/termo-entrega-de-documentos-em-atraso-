# 📄 Termo de Ciência – Não Entrega de Documento no Prazo

Template HTML gratuito e editável para uso em unidades escolares da rede pública ou privada.

---

## 📋 Sobre o documento

Este template foi desenvolvido para registrar formalmente a **não entrega de documentos pedagógicos dentro do prazo** estabelecido pela coordenação ou gestão escolar.

O termo serve como **comprovante formal** e pode ser utilizado em processos de acompanhamento docente e avaliação de desempenho individual.

---

## ✅ O que está incluído

- Cabeçalho adaptável com campos para nome da escola, secretaria/rede, unidade regional e programa/modalidade
- Identificação do professor(a), componente curricular e turma
- Registro do objeto da solicitação (campo aberto)
- Registro da comunicação prévia e do prazo descumprido
- Campo de implicações (editável conforme contexto)
- Campo para justificativa do(a) professor(a)
- Compromisso de regularização com nova data
- Campos de assinatura com nome, função e data
- Botão de impressão / exportação para PDF

---

## 🖊️ Como usar

1. Faça o download do arquivo `termo_ciencia_template.html`
2. Abra no navegador (Google Chrome, Edge ou Firefox)
3. Preencha os campos em branco com as informações da sua escola e do caso específico
4. Clique em **"Salvar como PDF / Imprimir"**
5. Em "Destino", selecione **"Salvar como PDF"**
6. Imprima em duas vias: uma para o arquivo da escola e uma para o(a) professor(a)

---

## ✏️ Como personalizar

O arquivo é um HTML simples, sem dependências externas. Para personalizar:

- Abra o arquivo em qualquer editor de texto (Bloco de Notas, VS Code, etc.)
- Localize os campos entre `<!-- -->` para identificar cada seção
- Substitua os textos conforme necessário
- Para adicionar o logo da sua escola, substitua a seção `.cabecalho-generico` por uma tag `<img>` com o caminho da imagem
